# improve — Claude Code Plugin

by [Musab Kara](https://linkedin.com/in/musab-kara-85580612a) · [GitHub](https://github.com/SkyWalker2506)

System improvement analyzer for Claude Code — analyzes any content (videos, articles, ideas) and extracts actionable improvements for your claude-config setup.

## Install

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/SkyWalker2506/claude-marketplace/main/install.sh) improve
```

Or via Claude Code native marketplace:

```bash
claude plugin install improve@musabkara-claude-marketplace
```
